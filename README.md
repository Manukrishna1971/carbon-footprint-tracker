\# 🌍 Carbon Footprint Tracker



A web-based application built using Flask that helps users estimate their carbon emissions based on daily activities. This project aims to raise awareness about environmental impact and encourage sustainable living.



\---



\## 🚀 Features



\* 🌱 Calculate carbon emissions based on user input

\* 📊 Simple and user-friendly interface

\* ⚡ Fast processing using Flask backend

\* 🧠 Basic emission estimation logic

\* 🖥️ Responsive web design using HTML templates



\---



\## 🛠️ Technologies Used



\* \*\*Python\*\*

\* \*\*Flask\*\*

\* \*\*HTML / CSS\*\*

\* \*\*OpenCV (cv2)\*\* \*(if used for detection features)\*

\## ⚙️ Installation \& Setup



\### 1️⃣ Clone the repository



git clone https://github.com/Manukrishna1971/carbon-footprint-tracker.git



\### 2️⃣ Navigate to project folder



cd carbon-footprint-tracker



\### 3️⃣ Install dependencies



pip install -r requirements.txt



\### 4️⃣ Run the application



python app.py



\---



\## 🌐 Usage



1\. Open your browser

2\. Go to: http://127.0.0.1:5000/

3\. Enter required inputs

4\. View your carbon emission results



\---



\## 📌 Example Calculation



Carbon emission is calculated using standard emission factors:



\* Petrol: 2.31 kg CO₂ per liter

\* Diesel: 2.68 kg CO₂ per liter
